### Projeto 1 – Consolidação de Dados Operacionais com Power BI

**Contexto**  
A operação utilizava múltiplos dashboards em Excel, aplicados individualmente para diferentes clientes. Embora funcionais localmente, essas planilhas geravam um gargalo para a gestão estratégica, exigindo consolidação manual de mais de 10 arquivos distintos.

**Desafio**  
Criar uma visão única, confiável e automatizada da operação, eliminando retrabalho, erros manuais e lentidão na análise de dados.

**Solução Desenvolvida**  
Projeto de Business Intelligence desenvolvido do zero em Power BI, com foco em centralização, padronização e geração de valor para tomada de decisão.

**Principais Atividades Técnicas**
- Tratamento avançado de dados com Power Query e linguagem M
- Criação de funções personalizadas para padronização automática de arquivos Excel
- Modelagem de dados relacional (Star Schema)
- Criação de medidas em DAX, incluindo uso de USERELATIONSHIP
- Consolidação de múltiplas fontes em um único modelo analítico

**Resultados**
- Visão estratégica centralizada da operação
- KPIs críticos de estoque, rupturas e excessos
- Análise de saúde do estoque e itens sem giro
- Monitoramento de qualidade, quebras e custos
- Eliminação total da consolidação manual

**Tecnologias**
Power BI | Power Query | Linguagem M | DAX | Modelagem de Dados

### Projeto 2 – VoteSys | Sistema Web de Votação Online (Full Stack)

**Contexto**  
Projeto acadêmico desenvolvido ao longo de um semestre, com o objetivo de aplicar, na prática, conceitos completos de desenvolvimento Full Stack, desde a arquitetura até o deploy final.

**Desafio**  
Desenvolver um sistema de votação online seguro, organizado e escalável, partindo do zero.

**Solução Desenvolvida**  
Sistema web completo de votação, com autenticação, controle de acessos, múltiplos perfis de usuário e documentação de API.

**Principais Funcionalidades**
- Autenticação segura com JWT (Spring Security)
- Recuperação e alteração de senha com envio automático de e-mails
- Arquitetura baseada em DDD
- Suporte a múltiplos usuários simultâneos
- API documentada com Swagger
- Deploy em ambiente cloud (Render)

**Stack Tecnológica**
- Back-end: Java | Spring Boot | JPA/Hibernate | Spring Security | JWT | PostgreSQL  
- Front-end: React | TypeScript | Redux Toolkit | Bootstrap | CSS

**Resultado**
Sistema 100% funcional, publicado online, consolidando o aprendizado prático em desenvolvimento Full Stack e ciclo completo de entrega de software.

### Projeto 3 – Reestruturação de Dashboard Avançado em Excel

Dashboard interno de gestão técnica que, inicialmente, utilizava inserção manual de dados e fórmulas diretas entre células, resultando em baixa escalabilidade, maior risco de erros e queda de performance.

- Redução do tempo de elaboração de relatórios de até 2 dias para meio período

- Dashboard dinâmico, confiável e escalável
- Aumento significativo de performance
- Apoio direto à tomada de decisão gerencial

### Projeto 4 – Sistema de Controle de Estoque de Ferramentas

**Contexto**  
Problema real identificado em cliente industrial: dificuldade na localização de ferramentas, falta de controle de estoque e perda significativa de tempo operacional.

**Desafio**  
Desenvolver uma solução tecnológica eficiente para controle, rastreabilidade e organização do estoque de ferramentas.

**Solução Desenvolvida**
Sistema de controle de estoque desenvolvido do zero, focado em eficiência operacional e confiabilidade dos dados.

**Funcionalidades**
- Cadastro completo de ferramentas
- Controle de entradas e saídas por máquina, operador e solicitante
- Alertas automáticos por e-mail para estoque mínimo
- Organização física mapeada por armários e localizações
- Importação e exportação de dados

**Impactos Alcançados**
- Redução de aproximadamente 80% no tempo de localização de ferramentas
- Redução de cerca de 50% na falta de ferramentas durante a produção
- Controle total do estoque e planejamento de compras mais assertivo

**Resultados em 5 Meses**
- Mais de 420 ferramentas cadastradas
- Mais de 150 movimentações registradas

**Tecnologias**
Java | JavaScript | HTML | CSS | IntelliJ IDEA | VS Code
